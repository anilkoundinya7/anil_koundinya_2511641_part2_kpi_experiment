# KPI Framework

## Primary KPI
Paid Conversion Rate

Reason:
This KPI directly measures how many users become paying customers.

## Supporting KPIs

1. Landing Page Visit Rate
2. Trial Start Rate
3. Onboarding Completion Rate
4. Average Revenue Per User
5. Average Revenue Per Converted User
6. Refund Rate
7. Support Ticket Rate
8. Average Engagement Score
9. Average Days To Convert

These KPIs evaluate the entire user journey from acquisition to revenue generation.
